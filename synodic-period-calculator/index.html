Lets one calculate the sidereal period of the planet (P) from the synodic period (S), and vice versa.

This calculator takes in positive decimal numbers. The result will not be calculated if an invalid input is entered.

If the result calculated is an infinity value or a negative value. "Ouch!" will be displayed as a result.
