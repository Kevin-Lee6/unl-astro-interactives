Help

This simulator models the rotation of stars around the North Star over time (both daily and seasonal motions are shown).

Input:

1. You can set the time through the clock or input the time through the text input.

2. The date can be set through the input for the day of the year or month picker.

3. If you would like to use current time and day. Hit Set to Sysmtem Clock.
